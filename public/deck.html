STEP 2 — 5-Tier 자동 분류
수집된 인플루언서를 즉시 5등급으로 분류
팔로워 수 + ER + 카테고리 + 콘텐츠 퀄리티를 규칙 기반 로직으로 종합 분석해서 자동 등급 배정.

분류 기준 (자동화 로직)
1차: 팔로워 수 → Tier 경계 자동 매핑
2차: ER(참여율) 기반 퀄리티 보정 — 팔로워 많아도 ER 낮으면 하향
3차: 콘텐츠 분석 — 최근 10개 영상/이미지 Vision 모델 분석, 카테고리 일치도 점수
4차: 이전 캠페인 성과 (DB 누적 후) → 실제 전환율 기반 재분류
핵심 인사이트
나노 (1K~10K)가 핵심 타겟 — 비용 0, 높은 ER, 대량 시딩 가능. 빠르게 성장 중인 크리에이터 집중
분류는 고정이 아님 — 3개월마다 성과 데이터 기반 재분류. 나노가 마이크로로 성장하면 자동 업그레이드
비용 대비 효과: 나노 50명 시딩 ROI > 메가 1명 (비용 0 vs 1,000만+)
🎯 퓨어 인플루언서 필터: 영상을 주기적으로 올리면서 조회수를 꾸준히 확보하는 크리에이터를 우선 식별
퓨어 스코어 (검증 점수): 콘텐츠 주기성 + 조회수 성장 추세 + 실제 참여율 + 봇 비율 추정 → 종합 0~100점. '찾기는 쉽고 검증이 어렵다' — 검증을 자동화 로직으로.
