난이도: 쉬움 | 시기: 즉시

난이도: 어려움 | 시기: 장기

IIS 지표 + 예산 어드바이저 + 역경매
인플루언서를 주식 인덱스처럼 종합 점수화(IIS). 예산 입력하면 최적 조합 자동 추천. 역경매로 효율 극대화.

📊 IIS 지표 (Influencer Index Score)

• 팔로워 비율 + 봇 제거 + 정제 데이터
• 주식 인덱스처럼 종합 점수화 (0~100)
• Pure Score 5항목: 업로드(30) + 조회비(25) + 참여(20) + 일관성(15) + 성장(10)
• 티어별 벤치마크 자동 비교
• 이퀄라이저: 파라미터 조절 → 실시간 점수 변동
현재 59명 DB에 적용 중

💰 예산 어드바이저

• 예산 입력 → 최적 인플루언서 조합 자동 추천
• ROAS 예측 (기존 캠페인 데이터 기반)
• Tier 배분 최적화 (나노 70% + 미드 25% + 메가 5%)
• 카테고리/국가별 단가 벤치마크
→ 메이커 의사결정 시간 90% 절감

🔄 역경매(입찰) 모델 기획중

• 기존: 찾기→제안 (광고주 주도)

4-1E-commerce Services · 이커머스 서비스
바이어 발굴 + 상세페이지 + 자적 유통
펀딩은 "신제품 마케팅 채널"로 재정의. 이커머스가 매출 주력. 바이어 발굴이 판매 실적과 직결.

🔍 LinkedIn 바이어 발굴

• 해외 바이어의 70~80%가 LinkedIn에 존재
• 자동 제안 발송 → 판매 실적 직결
• 고비즈코리아 바이어 20~30만 DB 접근 가능
• 스크롤링 자동화 검토 중
→ 바이어 발굴 + 물류 결합 = 풀스택 GTM

📄 상세페이지 생성 엔진

• 펀딩 상세페이지 = 고난이도 (자동화 어려움)
• 이커머스 상세페이지 = 패턴 있어서 가능
• 라쿠텐 상세페이지 자동화 즉시 검토
• 사진 집어넣고 글 조공 → 이미 3~4년 전에 있었음
• 우리는 더 고도화된 레벨
→ 입점 장벽 제거

🚀 자적 유통 시스템

• 한 번의 버튼으로 입점→판매→마케팅
• 예측 → 상세페이지 → 마케팅 → 입점 → 판매 = 원스톱
• 큐텐/라쿠텐/아마존 동시 확장
→ "보고서가 아니라 실행까지 연결해야 계약이 된다"

📈 펀딩 종료 후 로드맵 자동 제안

• 펀딩 종료 = 끝이 아니라 시작
• 자동 추천: 일본/대만/이커머스/D2C/바이어
• "이 제품은 일본 아마존에 가시면 됩니다" 수준
• 큐텐 키워드 (어드민에서 인기 키워드 제공)
• 라쿠텐 키워드 데이터 없음 = 기회
→ 백커 → 리피트 → LTV 1.8~2.5배
